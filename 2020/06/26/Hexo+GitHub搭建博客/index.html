准备工作

安装Git

​ Git 下载地址：https://gitforwindows.org/

安装Node.js

​ Node.js 下载地址：https://nodejs.org/en/

Hexo 本地搭建

在计算机中新建MyBlog文件夹用来存放博客相关文件

打开Git Bash，切换目录到MyBlog文件夹中

或者选择文件夹，右键单击Git Bash Here，在命令窗口执行命令

安装cnpm

防止使用国外的镜像因为网络延迟问题导致安装失败的问题

安装Hexo

安装可能有一些缓慢，当然这还是取决于你的带宽，这个过程中会出现的WARN提示不用理会。

hexo就安装好了，检查一下hexo -v是否安装成功

安装完成后目录

创建初始化目录

(不创建初始化目录会报错)

PS：注意，以后的操作都将在Hexo目录下进行

Hexo初始化

查看初始目录

node_modules：用于存放依赖包

public：存放生成的页面（第一次初始化时这个文件夹是不存在的）

scaffolds：生成文章的一些模板

source：用来存放你的文章

themes：用来存放下载的主题

_config.yml：博客的核心配置文件（设置主体、标题等属性）

接着需要执行一下cnpm install命令，要不下边的启动会提示命令不合法。

本地启动Hexo

直接访问 http://localhost:4000 ，本地博客搭建成功

在cmd命令窗口中创建一篇新的博客。

在你安装的博客目录下找到新创建的博客，用任一Markdown编辑软件编辑博客内容。

编辑完保存文件，在cmd窗口执行以下命令。

再次访问 http://localhost:4000 ，就可以在本地服务器看到你新编辑的博客

Hexo托管GitHub

GitHub官方网站：https://github.com

登录账号（没有账户注册一个即可），创建一个新的GitHub仓库

仓库名字Repository name为 账户名.github.io（如果名字不一样需要修改根目录配置文件），其他选项默认，单击创建按钮

创建完成后会自动分配一个仓库地址，默认为 https://github.com/账户名/仓库名.git

修改Hexo根目录下的主配置文件，添加创建的GitHub仓库地址

设置邮箱和用户名

执行上传命令

PS：上传需要输入用户名和密码

上传完成后访问域名即可访问到搭建的博客网站

附加：更改Hexo主题

下载主题：https://hexo.io/themes

选择一个主题，单击标签转到GitHub进行克隆下载

修改配置文件，Hexo文件夹根目录下的_config.yml文件，更改theme配置

butterfly为主题文件名

查看主题自述文件得知，这个主题需要 Pug & Stylus

打开搜索功能

更改主题其他选项配置参照自述文件修改即可

修改完成后，启动Hexo验证访问即可

验证访问：http://localhost:4000

标签 解释
title 【必需】文章标题
date 【必需】文章创建日期
updated 【可选】文章更新日期
tags 【可选】文章标签
categories 【可选】文章分类
keywords 【可选】文章关键字
description 【可选】文章描述
top_img 【可选】文章顶部图片
cover 【可选】文章缩略图(如果没有设置top_img,文章页顶部将显示缩略图，可设为false/图片地址/留空)
comments 【可选】显示文章评论模块(默认true)
toc 【可选】显示文章TOC(默认为设置中toc的enable配置)
toc_number 【可选】显示toc_number(默认为设置中toc的number配置)
auto_open 【可选】是否自动打开TOC(默认为设置中toc的auto_open配置)
copyright 【可选】显示文章版权模块(默认为设置中post_copyright的enable配置)
mathjax 【可选】显示mathjax(当设置mathjax的per_page: false时，才需要配置，默认false)
katex 【可选】显示katex(当设置katex的per_page: false时，才需要配置，默认false)
aplayer 【可选】在需要的页面加载aplayer的js和css,请参考文章下面的音樂配置
highlight_shrink 【可选】配置代码框是否展开(true/false)(默认为设置中highlight_shrink的配置)

创建友情链接页面

前往你的Hexo 博客的根目录
输入 hexo new page link
你会找到source/link/index.md这个文件

标签页面

进入您的hexo博客根文件夹

类型 hexo new page tags

你会找到 source/tags/index.md

修改 index.md

分类目录

  1. 进入您的hexo博客根文件夹。
  2. 类型 hexo new page categories
  3. 你会找到 source/categories/index.md
  4. 修改 index.md
