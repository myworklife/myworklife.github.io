VMware安装MacOS系统

发表于2020-07-10|更新于2020-11-15|Other
|阅读量:

1. 安装环境

  • Windows 10
  • VMware Workstation Pro 15

2. 所需工具

  1. VMware Workstation Pro 15

    下载地址：https://www.vmware.com/go/getworkstation-win

    序列号：AV3RH-AUX43-H84DP-8NZZX-XF8DA

  2. MacOS Unlocker for VMware

    Unlocker版本：https://github.com/BDisp/unlocker/releases

  3. MacOS光盘镜像

    系统版本：MacOS 10.13

    百度网盘：https://pan.baidu.com/s/1CxGLGnhMrlNU5OcRvgfZ7A 提取码：QWER

3. 准备工作

安装完成 VMware Workstation Pro 15 后，该软件有5个服务是自动启动的，使用 MacOS Unlocker for VMware 对 VMware Workstation Pro 15 进行破解，需要手动关闭这5个服务，因为破解工具无法对正在运行的文件进行修改。

1 . 打开运行窗口，输入services.msc打开服务控制台，手动关闭 VMware 的5个自动启动服务

  • VMware Authorization Service
  • VMware DHCP Service
  • VMware NAT Service
  • VMware USB Arbitration Service
  • VMware Workstation Server

2 . 将服务全部关闭后，运行VMware补丁工具

将Unlocker解压到英文目录下，找到 win-install.cmd 文件，单击右键菜单，选择以管理员方式运行即可。

运行完成后打开VMware Workstation Pro 15，可以看到选择客户机操作系统对话框中有Apple Mac OS X选项，则说明破解成功！

4. 安装MacOS

创建新的虚拟机

  • 硬件兼容性：Workstation 15.x
  • 客户机操作系统：Apple Mac OS X (MacOS 10.13)
  • 处理器配置：处理器数量 [2]，内核数量 [1]
  • 虚拟机内存：2048MB
  • 网络类型：使用网络地址转换 (NAT)
  • I/O 控制器类型：LSI Logic (推荐)
  • 磁盘类型：SCSI (推荐)
  • 选择磁盘：创建新虚拟磁盘 (40GB，单个文件)

开启此虚拟机

编辑虚拟机设置——CD/DVD (SATA)——使用 ISO 映像文件 (所有文件*.*)——开启此虚拟机

进入安装界面——以简体中文作为主要语言——设置安装MacOS (点击继续)——选择同意许可协议

此时还没有可用磁盘来进行系统安装

选择菜单栏——实用工具——磁盘工具——选择VMware Virtual SATA Hard Drive Media，点击抹掉，名称自定义，完成后关闭窗口

选择格式化的磁盘，点击继续，系统进入安装界面，大约15分钟完成，安装完成系统会自动重启。

5. 补充一下

启动系统前准备工作

！可选项，浏览了许多博客，大都会让修改这个文件，此教程并没有修改，虚拟机正常启动。

虚拟机安装建立后，别着急启动，还要更改这个虚拟机的配置文件。

找到安装虚拟机的位置，编辑以”.vmx”结尾的文件(虚拟机主配置文件)

在smc.present = "TRUE" 后面添加smc.version = "0"

安装VMware Tools

菜单栏——虚拟机——安装VMware Tools。

如果此方法无法进行安装，则可以使用下面的方法

运行完Unlocker中 win-install.cmd 文件后会自动生成一个Tools文件夹。ISO文件，插入虚拟机中运行安装即可。
